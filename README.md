# Minga
Dirt-simple markdown blog system built using Kotlin and KTOR.

## Blogging with Minga

1. Create content directory for *pages* and *articles*

```
/content_root
  /pages
     /slug
      media/
      index.md
  /articles
    /slug
      media/
      index.md
```

2. set *CONTENT_ROOT* ENV to index content

3. Run minga server
